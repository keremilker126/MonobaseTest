# 📝 MonoBase Note App - Showcase Project

Bu uygulama, **MonoBase SQLite Engine** altyapısını kullanarak geliştirilmiş, modern ve yüksek performanslı bir not yönetim sistemidir. Bir "Proof of Concept" (Kavram Kanıtlama) projesi olan bu uygulama, harici bir NoSQL motoruyla .NET MVC mimarisinin nasıl kusursuz bir şekilde entegre edilebileceğini gösterir.

---

## 🚀 Öne Çıkan Özellikler

* **MonoBase Entegrasyonu:** Veriler ilişkisel tablolar yerine, MonoBase API aracılığıyla dinamik JSON koleksiyonlarında saklanır.
* **Modern UI/UX:** Inter font ailesi, Glassmorphism efektleri ve FontAwesome ikonları ile zenginleştirilmiş, kullanıcı dostu arayüz.
* **Asenkron Mimari:** Tüm veritabanı ve API işlemleri `Task` tabanlı asenkron yapıda gerçekleştirilir (Non-blocking UI).
* **Responsive Tasarım:** Bootstrap 5 kullanılarak tüm cihazlarla (Mobil, Tablet, Masaüstü) tam uyumlu hale getirilmiştir.
* **Güvenli İletişim:** API işlemlerinde her kullanıcıya özel API Key doğrulama mekanizması kullanılır.

---

## 🛠️ Teknik Stack

* **Framework:** ASP.NET Core 8.0 (MVC)
* **Veri Motoru:** [MonoBase SQLite Engine](https://github.com/keremilker126/MonoBase) (Harici Servis)
* **Frontend:** HTML5, CSS3 (Custom), Bootstrap 5, JavaScript (ES6+)
* **İletişim:** HttpClient Factory & JSON Serialization
* **İkon Seti:** FontAwesome 6.5.1
